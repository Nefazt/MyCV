Que hago ?

Desarrollo software y hardware, construyo dispositivos con MCUs que pueden ayudar con diferentes tareas, entre ellos un dispositivo inalambrico que genera una alarma en el caso de presentarse una inmersion accidental en una piscina y se pueda atender de manera oportuna, la realice bajo lenguaje C ++ y analisis de datos obtenidos por los sensores en Python. Tambien desarrolle un dispositivo de domotica con interfaz web para el cuarto de maquinas de una piscina y jacuzzi. Me encanta el estilo limpio y elegante, tengo experiencia en el desarrollo de sitios web modernos ademas de conocimiento y manejo de diferentes lenguajes de programación.
